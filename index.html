ยินดีต้อนรับเข้าสู่คริสตจักรพระกิตติคุณสมบูรณ์ธารทิพย์

คริสตจักร มีสมาชิก รวมกันทั้ง เด็ก อนุชน ผู้ใหญ่ รวมกันประมาณ 50 คน

เมื่อก่อน คริสตจักร มีการสอนเด็กระวี แต่เดี๋ยวนี้ไม่มีสอนแล้ว

ลำดับ ชื่อ หน้าที่ต่างๆ
1. ศจ. ปกาศิต ชัยงาม หน้าที่เทศนา
2. คุณวิลรี่ ชัยงาม ผู้นำนมัสการ
3. คุณฟุ้น / คุณลำหยวน / คุณจ๋าย คณะผู้ปกครอง
4. คุณสายบัว / คุณใส / คุณธนันญา เหรัญญิก
5. คุณมานิตา / คุณจันทรา / คุณอรไพลิน นักดนตรี
6. คุณโรสมารินทร์ ฝ่ายธุรการ
7. คุณสายบัว ผู้ประกอบอาหาร
